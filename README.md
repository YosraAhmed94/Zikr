# Zikr
Zikr is an open-source iOS app which remind the muslim with one of the most easy and important ibada which is Zikr.

# Features
 - Hisn-ul-muslim book.
 - An english translated and franco-arabic prononciation for athkar.
 - Alarm for reminding athkar.
 - Playing audio for athkar.
 - Add favorites athkar.
 - Add custom athkar.

## Dependencies
- The book is got from [Hisn-ul-muslim API](https://www.hisnmuslim.com).
